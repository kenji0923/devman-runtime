# devman-runtime

Per-language runtime libraries for bridges generated by
[devman-gen](https://github.com/kenji0923/devman-gen). The generator is
language-agnostic; each language target has its runtime in a subdirectory:

- `python/` — the Python runtime, published to PyPI as
  [`devman-runtime`](https://pypi.org/project/devman-runtime/)
  (tags `v*` trigger the publish workflow).
- `cpp/` — (planned)

See `python/README.md` for the Python package documentation.
